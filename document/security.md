# Security Policy

## Supported Versions

The following versions of the **shopapp-backend** project are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately.

**How to report:**
Please report vulnerabilities by contacting the project maintainers directly or opening a confidential issue if the platform supports it. Do not disclose vulnerabilities publicly until they have been addressed.

**Expectations:**
- We aim to acknowledge reports within 48 hours.
- Updates on the vulnerability status will be provided regularly.
- You will be notified when a fix is released.
